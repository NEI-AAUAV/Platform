# External libraries



# Typing effect

This effect is provided by **React Typist** component.

NPM: 

https://www.npmjs.com/package/react-typist

Usage:

```react
import Typist from 'react-typist';

const MyComponent => () => {
    
    return (
    	<h1>
        	<Typist>My typing animation title</Typist>
        </h1>
    )
    
}
```
